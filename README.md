# SV Infonet — Legal Policies (SV eCAF App)

This repository hosts the Privacy Policy and Terms & Conditions for the **SV eCAF** Android application (`com.svinfonet.ecaf`) published on the Google Play Store by **SV Infonet Solution Private Limited**.

## Pages

| Page | URL |
|------|-----|
| Home | `https://YOUR-USERNAME.github.io/svinfonet-policies/` |
| Privacy Policy | `https://YOUR-USERNAME.github.io/svinfonet-policies/privacy-policy.html` |
| Terms & Conditions | `https://YOUR-USERNAME.github.io/svinfonet-policies/terms.html` |

> Replace `YOUR-USERNAME` with your actual GitHub username after publishing.

## How to Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** → **Pages** (left sidebar)
3. Under **Source**, select `Deploy from a branch`
4. Choose branch: `main` and folder: `/ (root)`
5. Click **Save**
6. Wait 1–2 minutes, then visit the URL above

## Files

- `index.html` — Landing page with links to both policies
- `privacy-policy.html` — Full Privacy Policy
- `terms.html` — Full Terms & Conditions

## Update Placeholders

Before publishing, update these placeholders in the HTML files:
- `privacy@svinfonet.com` → actual privacy email
- `legal@svinfonet.com` → actual legal/contact email
- `www.svinfonet.com` → actual website (if available)
- `[Your Registered Address]` → company registered address
- `[Company's Registered State/City]` → for governing law clause
